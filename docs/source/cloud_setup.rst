Local cloud setup
=================

You can manage your local clouds by using the ``cloud`` command.
To see what it has to offer, try

.. code-block:: bash

   pyrrowhead cloud --help

The ``cloud`` command has seven subcommands to manage multiple local clouds at the same time.
